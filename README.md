# Simple Calculator in C

A basic command-line calculator that performs arithmetic operations.

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## How to Use
1. Compile: `gcc calculator.c -o calculator`
2. Run: `./calculator`
3. Enter first number, operator, then second number
## Example
Enter a number: 10
Enter operator: +
Enter a number: 5
answer is: 15.000000
